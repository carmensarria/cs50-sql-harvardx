# 36 Views

This exercise is part of Week 0 of CS50's Introduction to Databases with SQL.

## Description

36 Views consists of multiple SQL queries focused on exploring structured data and extracting specific information from relational datasets.

## File included

- 36-views.sql

## SQL Concepts Used

- SELECT queries with multiple columns
- Filtering with WHERE and LIKE
- Aggregation (COUNT, MAX, AVG)
- Sorting results with ORDER BY
- Limiting results with LIMIT
- Column aliasing with AS
- Data analysis on artistic datasets
