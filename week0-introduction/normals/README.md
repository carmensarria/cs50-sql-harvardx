# Normals

This exercise is part of Week 0 of CS50's Introduction to Databases with SQL.

## Description

Normals focuses on analyzing ocean temperature data across different latitudes, longitudes, and depths using SQL queries.

## File included

- normals.sql

## SQL Concepts Used

- SELECT queries with multiple columns
- Filtering with WHERE and BETWEEN
- Aggregate functions (MIN, MAX, AVG, COUNT)
- Rounding numerical results
- Sorting with ORDER BY
- Limiting results with LIMIT
- Geographic data analysis using latitude and longitude
