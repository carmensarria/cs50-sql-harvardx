# Players

This exercise is part of Week 0 of CS50's Introduction to Databases with SQL.

## Description

Players focuses on analyzing structured baseball player data, including biographical information, physical attributes, and career history.

## File included

- players.sql

## SQL Concepts Used

- SELECT queries with multiple columns
- Filtering with WHERE, AND, OR
- NULL value handling (IS NULL)
- Sorting with ORDER BY (multiple columns)
- Aggregation functions (COUNT, AVG)
- Date filtering
- Column aliasing with AS
