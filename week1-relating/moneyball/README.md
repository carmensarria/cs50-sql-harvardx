# Moneyball

This exercise is part of Week 1 of CS50's Introduction to Databases with SQL.

## Description

Moneyball is a data analysis exercise based on baseball statistics. The goal is to explore player performance, salaries, and team efficiency using SQL queries over relational data.

## Files included

- moneyball.sql

## SQL Concepts Used

- SELECT queries
- JOIN operations
- Aggregation functions (COUNT, AVG, SUM)
- Filtering with WHERE
- Sorting with ORDER BY
- Subqueries
- Data analysis on sports datasets
