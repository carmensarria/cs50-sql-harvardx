# DESE

This exercise is part of Week 1 of CS50's Introduction to Databases with SQL.

## Description

DESE is an educational data analysis exercise that involves querying structured datasets related to schools, performance metrics, and statistical information.

The goal is to extract insights using SQL queries over relational data.

## File included

- dese.sql

## SQL Concepts Used

- SELECT statements
- Filtering with WHERE
- Aggregation functions (COUNT, AVG, etc.)
- GROUP BY (if applicable)
- Sorting with ORDER BY
- Data analysis on structured datasets
- Working with relational data
