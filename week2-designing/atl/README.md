ATL: Airport Database Design  
This exercise is part of Week 2 of CS50's Introduction to Databases with SQL.

Description
ATL focuses on designing a relational database for Hartsfield-Jackson International Airport, one of the busiest airports in the world.

The goal is to model passengers, airlines, flights, and check-ins using a structured SQL schema based on real-world requirements.

test.sql

SQL Concepts Used
CREATE TABLE statements
Primary keys and foreign keys
NOT NULL constraints
CHECK constraints

Relational database design
One-to-many relationships
