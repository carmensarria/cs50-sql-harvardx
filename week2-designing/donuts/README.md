# Donuts

This exercise is part of Week 2 of CS50's Introduction to Databases with SQL.

## Description

Donuts focuses on designing and normalizing a relational database for a fictional donut shop: Union Square Donuts.

The database stores information about:

- Ingredients
- Donuts
- Customers
- Orders
- Relationships between donuts and ingredients
- Relationships between orders and donuts

This exercise emphasizes many-to-many relationships, normalization, and relational database design.
